# نظام متابعة المهام الدراسية (Laravel)

هذا المشروع هو تطبيق ويب بسيط تم تطويره باستخدام إطار العمل **Laravel**،
ويهدف إلى مساعدة الطلاب على تنظيم ومتابعة المهام الدراسية ومواعيد التسليم.

---

## فكرة المشروع
- إضافة مهام دراسية
- عرض جميع المهام
- عرض تفاصيل كل مهمة
- تعديل وحذف المهام
- حساب إحصائيات المهام (منجزة – قيد الانتظار – متأخرة)

---

## ملاحظات مهمة
- لا يستخدم قاعدة بيانات (حسب متطلبات المشروع)
- يتم تخزين البيانات باستخدام **Session**
- يعتمد على نمط MVC في Laravel
- الواجهات مبنية باستخدام Blade و CSS فقط

---

## التقنيات المستخدمة
- Laravel
- PHP
- Blade Template Engine
- HTML / CSS
- Session Storage

---

## تشغيل المشروع محليًا

1. التأكد من تشغيل XAMPP أو أي سيرفر محلي يدعم PHP.
2. فتح مجلد المشروع باستخدام Terminal أو Command Prompt.
3. تشغيل السيرفر المحلي باستخدام الأمر:

```bash
php artisan serve
فتح المتصفح والدخول على الرابط التالي:
